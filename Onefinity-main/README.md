# Onefinity – CNC Industrial Control Interface

A modern CNC machine control interface (React, TypeScript, Three.js, Express, Socket.IO, GRBL).

- **Quick start (local):** `npm run dev` (frontend) and `npm run backend` (backend). See [docs/README.md](docs/README.md).
- **Docker:** `docker compose up -d` (frontend: 6080, backend: 6070, Redis: 6060). Optional log viewer: `docker compose --profile tools up -d` (Dozzle: 6040).
- **Documentation:** All guides and references are in the [docs/](docs/) folder.
